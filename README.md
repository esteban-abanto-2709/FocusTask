# 🎯 Focus Task

> **"No pienses qué hacer, simplemente hazlo"**

Una aplicación inteligente de gestión de tareas que elimina la parálisis por análisis. En lugar de perderte navegando entre cientos de tareas, Focus Task te entrega la tarea más importante con un solo clic.

## 🚀 El Problema Que Resolvemos

¿Te has encontrado alguna vez con una lista interminable de tareas sin saber por dónde empezar? ¿Pierdes tiempo valioso decidiendo qué hacer en lugar de realmente hacerlo?

Focus Task elimina este friction cognitivo mediante un sistema de **recomendación inteligente** que evalúa tus tareas y te presenta la más importante para tu contexto actual.

## ✨ Características Principales

### 🎯 Core Features (Fase Actual)

- **Gestión Intuitiva de Tareas**: Crea, visualiza y organiza tareas con prioridades
- **Interfaz Moderna**: Diseño glassmorphism con gradientes dinámicos
- **Sistema de Prioridades**: Alta, Media, Baja con indicadores visuales
- **Arquitectura Escalable**: Context API con TypeScript para máxima type-safety

### 🔮 Próximas Características

- **🎲 "Dame una Tarea"**: Algoritmo inteligente que selecciona la tarea óptima
- **📊 Gestión de Proyectos**: Organiza tareas por proyectos específicos
- **🤖 Importación IA**: Genera tareas masivamente desde JSON con prompts de IA
- **📱 Experiencia Móvil**: PWA responsive para productividad en movimiento
- **☁️ Sincronización en la Nube**: Acceso multiplataforma con cuentas de usuario

## 🚀 Instalación y Desarrollo

### Prerrequisitos

- Node.js 18+
- npm o yarn

### Setup Rápido

```bash
# Clonar el repositorio
git clone https://github.com/tuusuario/focus-task.git
cd focus-task

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Abrir en http://localhost:5173
```

### Comandos Disponibles

```bash
npm run dev      # Desarrollo con hot reload
npm run build    # Build de producción
npm run preview  # Preview del build
npm run lint     # Linting con ESLint
```

## 🎨 Flujo de Trabajo con IA

Una característica única de Focus Task es su integración con herramientas de IA para generación masiva de tareas:

1. **Consulta a tu IA favorita**: Describe tu proyecto
2. **Usa nuestro template**: Solicita las tareas en formato JSON específico
3. **Importa instantáneamente**: Un clic y todas las tareas están listas
4. **Comienza a trabajar**: Presiona "Dame una Tarea" y actúa
